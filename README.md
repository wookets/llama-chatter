# llama-chatter

A simple repo that has scripts that will generate chatter amongst LLMs.

## Dependencies

* Install pipx.
* This repository assumes you are using LLMs behind ollama compatible apis.

## Usage

```sh
poetry run python main.py
```

## Setup

```sh
brew install pipx
pipx ensurepath
# refresh / re-source your omyzsh
pipx install poetry
poetry install
poetry run python main.py
```
